# Naming Rules

## Document File Names

All documentation files **MUST** use English names.

- ✅ Good: `riskIndex.md`, `DESIGN.md`, `INPUT_GUIDE.md`
- ❌ Bad: `风险指数.md`, `设计文档.md`, `输入指南.md`

## Other Files

Code files, configuration files, and data files should also follow English naming conventions where appropriate.
